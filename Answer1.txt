Q1:
Rounding of Grades .. if user grades is 88 or more round it to 90 if the grade is 87 keep it .. if user grade 63 round it to 65 if 62 keep it .. so itâ€™s on factorial of 5 where 3 grades or more round .. 2 or less keep the grade
A1:
public class HW1 {
    static int roundup(int grade){
        int newgrade=grade;
        if((newgrade+1)%5==0||(newgrade+2)%5==0){
            if(grade+1%5==0){
                newgrade++;
            }else{
                newgrade+=2;
            }
        }
        return newgrade;

    }
    public static void main(String[] args) {
        int grade=roundup(88);
        System.out.println(grade);
    }


}

Q3:
- pass a name to the function that return the count of vowels in that name
- print the result

A3:
public class HW3 {
    static int vowels(String name){
        int vowels_num=0;
        String vowels="aeiou";
        for(int i =0;i<name.length();i++){
            char current_char=name.charAt(i);
            for(int j=0;j<vowels.length();j++){
                if(current_char==vowels.charAt(j)){
                    vowels_num++;
                }
            }
        }

        return vowels_num;
    }
    static int max(int num1,int num2){
        return num1>num2 ? num1: num2;
    }
    public static void main(String[] args) {

        String name="ali alabdrabalrasol";

        System.out.println(vowels(name));
        System.out.println(max(10,4));

    }
}
